# My AI & Machine Learning Journey 🚀

Welcome to my AI and Machine Learning repository! This repository serves as a personal log of my progress, containing the code, notes, and projects I've built while mastering Data Science, Machine Learning, Deep Learning, and NLP.

## 📈 Detailed Progress Log

I have covered an extensive curriculum, diving deep into Python, Data Science, Statistics, and Machine Learning. Below is a detailed breakdown of the concepts I have successfully completed and implemented so far:

### 1. Python Programming Fundamentals & Advanced Concepts
*   **Basics:** Environment setup, Variables, Data Types, Operators.
*   **Control Flow:** Conditional statements (if/elif/else), Loops.
*   **Functions:** Lambdas, Map, Filter, Closures, and Decorators.
*   **File Handling:** Working with file paths and file operations.
*   **Modules & Packages:** Standard Library overview, importing and structuring projects.
*   **Exception Handling:** Try, except, else, finally blocks, and Custom Exceptions.
*   **Object-Oriented Programming (OOP):** Classes, Objects, Inheritance, Polymorphism, Encapsulation, Abstraction, Magic Methods, Operator Overloading.
*   **Advanced Concepts:** Iterators, Generators, Memory Allocation & Garbage Collection.

### 2. Databases & Web Frameworks
*   **Databases:** SQLite3 CRUD operations.
*   **Logging:** Multi-logger implementations and real-world logging use cases.
*   **Web Frameworks:** Building Machine Learning Web Apps using Streamlit and Flask.
*   **Parallelism:** Multithreading, Multiprocessing, Thread Pools, Process Pools, and Web Scraping use cases.

### 3. Data Science & Exploratory Data Analysis (EDA)
*   **Libraries:** NumPy for arrays, Pandas for DataFrames/Series.
*   **Data Manipulation:** Reading from various sources and comprehensive data manipulation.
*   **Data Visualization:** Matplotlib and Seaborn for insightful graphs.
*   **Feature Engineering:**
    *   Handling missing values and outliers.
    *   Handling Imbalanced Datasets (using SMOTE).
    *   Data Encoding (Nominal, One-Hot Encoding, Label, Ordinal, Target-Guided).

### 4. Statistics & Probability
*   **Descriptive Statistics:** Measure of Central Tendency, Dispersion, Variance, Standard Deviation, Histograms, Percentiles, Quartiles, Covariance, and Correlation.
*   **Probability:** Addition/Multiplication Rules, PDF/PMF/CDF, Bayes' Theorem.
*   **Distributions:** Bernoulli, Binomial, Poisson, Normal/Gaussian, Uniform, Log-Normal, Power Law, Pareto.
*   **Inferential Statistics:** Central Limit Theorem, Hypothesis Testing, Z-Test, T-Test (and T-Stats), Chi-Square Test (Goodness of Fit), ANOVA (Assumptions, Types, Partitioning).
*   **Errors & Intervals:** Type 1 and Type 2 Errors, Confidence Intervals, Margin of Error.

### 5. Machine Learning (Supervised Learning)
*   **Core Concepts:** Instance vs. Model-based learning, Hyperplanes, Cost Functions, Convergence Algorithms, Overfitting vs. Underfitting.
*   **Linear Regression:** Simple and Multiple Linear Regression (OLS, Assumptions, Pipeline, Pickling).
*   **Advanced Regression:** Polynomial Regression, Ridge, Lasso, and Elastic Net (with Cross-Validation).
*   **Logistic Regression:** OVR, Imbalanced Datasets, ROC/AUC, Mathematical Intuition.
*   **Support Vector Machines (SVM):** Soft vs. Hard Margins, Support Vector Classifiers (SVC), Support Vector Regressors (SVR), and Kernels (Math & Implementation).
*   **Model Optimization:** Performance Metrics (MSE, MAE, RMSE), Grid Search, Randomized Search CV for Hyperparameter Tuning.
*   **MLOps:** End-to-End ML Project Implementation and Deployment to AWS.
